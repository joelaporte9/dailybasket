Preface

Welcome to the daily bakset. The name "basket" comes from the name I use for everything. The goal for this website was to create a space to post all thoughts, photos and more, a basket for all of my stuff.

The idea behind this website came from the fact that I don’t use social media to share about my life. For some reason, I just don't like to share on big social media platforms like Instagram, only the occasional story… There is more to my life than just instagram stories, so here is where you can catch up on my life.

I graduated from Elms College in 2023 with a bachelors in Computer Science and a minor in Cybersecurity, I also hold a CompTIA Security+ certification. Now I live and work in San Diego as a Software Development Engineer in Test (SDET), I create and test software. With that said, I decided to take it upon myself to design, build and now maintain this website. It has been made using Python + Django web framework with PostgreSQL for the backend and simple HTML and CSS for the frontend.

I plan to follow the POSSE model, publish own site, syndicate everywhere/elsewhere. This allows me to keep full control of my blogs and photos while tyring to get engagment from heavily trafficed sites, like IG. So if you came to this site via a self promo, I'm sorry...

~ This site is dedicated to anyone who helped me along the way. ~
